Update Hexo 3.1.1

发表于 2015-08-28   |  

在当前静态博客系统中，Hexo 配合 Github Page 是一个比较方便的搭建方式。

之前一直使用的 Hexo2.x 和 Github 配合记些笔记，换系统(opensuse12->13)之后，Hexo 版本也升级了。迁移比较麻烦，正好从头安装一遍3.x版本，换一个简明的主题。

下面记录一些新版本的特点和遇到的问题以及流程。

若在下面的步骤中卡住了，请查看最新的官网安装指南，或者自行搜索解决方法

install git

  1. 安装，通常在Linux和osx系统上已安装

  2. 配置用户名和电子邮件

  3. 生成sshkey，并测试(可以根据 github官网的帮助 来完成)

install nodejs, hexo

由于G5W的原因，无法从官网下载，下面从github上下载安装程序，并安装

install hexo

Hexo 本次更新到 3.1 新版本后，安装方式也略微有一些改变
$ npm install -g hexo-cli --registry http://registry.cnpmjs.org
命令后面的参数表示使用国内镜像

setup

how to use it

基本的使用方法，在官网的中文文档中十分详细，请自取。

看完这些基本的说明以后，就可以尝试写一篇文章了。然后再尝试不同的主题，比如下面将要介绍的这个。

theme — NexT

这是一个国人写的主题，简介明了，可以很方便地上手

直接贴链接： http://theme-next.iissnan.com/ 一步一步根据教程来就可以了
