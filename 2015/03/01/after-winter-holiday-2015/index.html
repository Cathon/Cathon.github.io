小结与展望

发表于 2015-03-01   |  

冬去春来。
寒假过去，新学期到来。
是该总结一下了。

##before

自从了解到 ctf 这个比赛之后，一直在找这方面的资源。
寒假也做了不少题。通过合天网安的平台也学了不少东西。
假期中对 android 也有所了解，虽然还没不知道大多数的控件类库等等，也没有完成一个简单应用。
不过也算基本接触到了。

情人节安全宝的活动中那道 android 的题经过努力也能够成功 hack 了。
android 反编译也学了个七七八八，当然实战经验基本没有。

假期学习的隐写术利用了很多工具，绝大多数都能在 linux 环境下使用，这也在某些方面证明了开源软件的优势。
但是，工具也是人写出来的。
期间遇到很多例子是必须自己手写解决的。

但是 python 不大会写。。。所以，接下来要好好练习 python
至少读一边官方手册（目前也许做不到）

仔细想一想，寒假学得东西太零散，也没有好好整理，正好这学期利用兴趣小组的教学机会巩固一下。

##now

这几天还在做 ctf
据说 boston key party 是北美最具影响力的比赛之一，于是参加了下。（落笔时比赛还未结束）

比赛的分类如下：
Boston Key Party 2015 » Pwning Reversing Crypto “School Bus”

以我的水平也就做做最后一项。估计就是学生水平吧，主要内容是 php 代码审计。
一共十题，pass 7个，代码审计的那几个（在和群友的互相帮助下）基本搞定，知识点也是我未曾了解的。
加密和逆向的题我也只能看看题目了，至于二进制完全不懂。

##after

明天周一，要上课了。
这学期要修的是： 算法，编译原理，计组，操作系统。
好像都是重头戏啊

另外，还有<20d，就要完成一个Java项目
参加 xctf 今年的联赛
ctf 的教学，想举办一场校内赛
Java 学习（因为我决定用 Java 来写算法）
Android，无线端的破解
python（排在这么后面（～_～））

好了，我该去睡觉了～
